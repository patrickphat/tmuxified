<h1 align="center">Tmuxified</h1>

<p align="center">
  <b>A simple configuration for tmux.</b>
</p>

<br/>


## Installation  

```bash
cd
git clone git://github.com/zaiste/tmuxified.git

# Link tmux.conf
ln -sfn ~/tmuxified ~/.tmux
ln -sfn ~/tmuxified/tmux.conf ~/.tmux.conf
```
